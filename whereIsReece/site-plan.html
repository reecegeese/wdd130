Overview

Purpose

To answer the age old question Where is Reece? From daily classes, hitting the gym, or going out on the town Reece could be in a lot of places. However, he is consistent so this site will lessen the possibilities

Audience

Anyone looking for Reece at any given time. Somtimes you need to talk in person, someimes you're in the area. Whatever happens make sure you find him first
